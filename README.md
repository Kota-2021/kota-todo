# **# kota-todo**

## **リアルタイム通知・非同期処理を備えたタスク管理 Web API**

## **1. プロジェクトの核となる2つの特徴**

本プロジェクトは、単なるCRUDに留まらず、実務レベルの負荷と信頼性を想定して開発しました。

* **非同期メッセージング (SQS)**: 重い処理をWorkerに逃がすことで、APIレスポンスをミリ秒単位で維持する疎結合設計。  
* **リアルタイム通知 (WebSocket + Redis)**: Goの並行処理とRedis Pub/Subを活用した、スケーラブルな通知基盤。

## **2. 技術スタック**

| カテゴリ | 技術 | 選定理由 |
| :---- | :---- | :---- |
| **Backend** | Go (Gin) | 高い並行処理能力と実行速度。 |
| **Infra** | AWS (Fargate) | 運用負荷を抑えたスケーラブルな実行環境。 |
| **Messaging** | SQS / Redis | 非同期処理による高可用性とリアルタイム性の両立。 |
| **IaC / CI/CD** | Terraform / GitHub Actions | 環境の100%再現とデプロイの完全自動化。 |
